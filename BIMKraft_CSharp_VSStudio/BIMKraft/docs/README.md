## About BIMKraft Tools / Über BIMKraft Tools

**English:**
BIMKraft Tools is a comprehensive suite of productivity tools for Autodesk Revit, designed to streamline workflows and enhance parameter management. The tools are accessible through a dedicated BIMKraft tab in the Revit ribbon. Developed by Maria Simon (bimkraft.de).

**Deutsch:**
BIMKraft Tools ist eine umfassende Suite von Produktivitätswerkzeugen für Autodesk Revit, die entwickelt wurden, um Workflows zu optimieren und die Parameterverwaltung zu verbessern. Die Tools sind über eine dedizierte BIMKraft-Registerkarte im Revit-Menüband zugänglich. Entwickelt von Maria Simon (bimkraft.de).

---

## Current Tools / Aktuelle Tools

### Parameter Tools Panel

#### Parameter Pro
**EN:** Advanced parameter management with presets, category merging, and batch operations
**DE:** Erweiterte Parameterverwaltung mit Vorlagen, Kategoriezusammenführung und Stapeloperationen

#### Parameter Transfer Pro
**EN:** Advanced parameter value mapping and transfer tool with category-based mappings and preset support
**DE:** Erweitertes Tool zur Parameterwert-Zuordnung und -Übertragung mit kategoriebasierten Zuordnungen und Vorlagenunterstützung

### Workset Tools Panel

#### Workset Assignment Tools
**EN:** Suite of tools for automatically assigning elements to appropriate worksets (Grids, Levels, References, Architectural, Steel, Reinforcement, Structural Shell)
**DE:** Suite von Tools zur automatischen Zuweisung von Elementen zu entsprechenden Arbeitsbereichen (Raster, Ebenen, Referenzen, Architektur, Stahl, Bewehrung, Rohbau)
